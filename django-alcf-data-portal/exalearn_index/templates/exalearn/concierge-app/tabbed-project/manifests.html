Custom Subsets

Below is a list of your Custom Subsets, each one tracking data associated with ExaLearn search results.

You can create Subsets from the search page by filtering results and 'checking out' a subset of those results. After that, you can get the data in your subset a couple of different ways:

  • Download a BDBag, a small zip file which tracks file references and metadata.
  • Transfer the data directly to a remote location using Globus.
